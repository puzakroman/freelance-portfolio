# Freelance Portfolio

A collection of professional automation tools and database management scripts developed for freelance projects.

## Included Projects

### 1. Web Data Extraction Tool (Python)
- **Location:** `/portfolio/scraper`
- **Purpose:** Automated web scraping with robust error handling and CSV export.
- **Tech Stack:** Python, BeautifulSoup, Requests.

### 2. Database Migration Utility (PHP)
- **Location:** `/portfolio/db_importer`
- **Purpose:** High-performance CSV to MySQL/MariaDB data migration with transaction support.
- **Tech Stack:** PHP (PDO), MySQL.
